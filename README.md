# Typescript Express API Starter

## Install

```bash
git clone git@github.com:GainorB/typescript-node.git name-of-folder
cd name-of-folder
yarn
```

## Development

```bash
yarn start:dev
```

## Deploy to Heroku

```bash
yarn deploy
```

## Production

```bash
yarn build
yarn start
```

## Environment Variables

```
PORT=4000
NODE_ENV=dev
```

## Tech Stack

1. Typescript
2. REST API
3. Express.js
4. Node.js
5. Winston logger
